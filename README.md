# UE5项目：RPG开发日志

本项目记录了一款基于 Unreal Engine 5 的游戏demo开发过程

---

##  开发进度回顾

###  2026-04-12 | 核心运动系统
* [x] **角色基础控制**：实现 8 向移动、平滑转向逻辑。
* [x] **姿态系统**：实现基础蹲下 (Crouch) 与站立切换。
* [x] **程序化倾斜**：根据移动速度和转向角度，实现角色的动态身体倾斜。
* [x] **跨越系统 (Vaulting)**：基于射线检测与物理位置偏移，实现角色自动跨越障碍物。

###  2026-04-14 | 物理与基础数值
* [x] **暗杀系统**：实现潜行靠近后的交互逻辑。
* [x] **布娃娃系统 (Ragdoll)**：整合物理资产，实现角色死亡或特定状态下的物理表现。
* [x] **基础 UI 框架**：
    * **血量条 (HP Bar)** 动态绑定。
    * **体力条 (Stamina Bar)** 逻辑构建。

###  2026-04-15 | 进阶系统与数值反馈
* [x] **奔跑机制**：实现加速奔跑，并同步 **体力衰减 (Stamina Decay)** 算法。
